# 재판랭 (JaepanLang)

**한글 난해한 프로그래밍 언어 재판랭 1.5!**
- 적당히 난해하고
- 대충 이해는 가고
- 웃기기도 하고
- 생각보다 파워풀하다.

## 기본 개념
- Brainfuck을 한글 감성으로 재해석
- 포인터, 메모리 조작, 루프, 연산 기능 제공

## 명령어
| 명령어 | 기능 |
|:---|:---|
| 재판업 | 포인터 오른쪽 이동 |
| 재판다운 | 포인터 왼쪽 이동 |
| 재 | 현재 칸 +1 |
| 관 | 현재 칸 -1 |
| 퉤 | 현재 칸 출력 |
| 츄릅 | 입력받기 |
| 박 | 루프 시작 |
| 재판 | 루프 종료 |
| 마딱 | 현재 += 옆 칸 |
| 다딱 | 현재 -= 옆 칸 |
| 과 | 현재 == 옆 칸 비교 |
